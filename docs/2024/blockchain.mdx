- 介绍 Merkle Tree 的性质、优点，为什么用 Merkle Tree 存（实现 SPV、Merkle Proof）
- 为什么以太坊要将 MT 改成 MPT（这个问题比较复杂，建议读者去 B 站北大肖臻老师的区块链课中寻找答案）
- 介绍比特币的 UTXO
- 比特币查询余额只能从头开始遍历整条链吗，有没有高效方法（创建 UTXOSet 缓存）
